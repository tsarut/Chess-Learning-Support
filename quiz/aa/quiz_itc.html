แบบทดสอบหมากรุกสากล ไทย และจีน
1. หมากรุกประเภทใดไม่สามารถใช้กระดานของหมากรุกชาติอื่นมาแทนได้
1.1) หมากรุกสากล
1.2) หมากรุกไทย
1.3) หมากรุกจีน
1.4) ใช้กระดานแทนกันได้ทุกชาติ
2. กลุ่มหมากในข้อใดที่มีวิธีการเดินเหมือนกันทุกประการ
2.1) King / ขุน / ตี่
2.2) Knight / ม้า / เบ๊
2.3) Rook / เรือ / กือ
2.4) Pawn / เบี้ย / จุก
3. กลุ่มหมากในข้อใดที่มีวิธีการเดินต่างกันโดยสิ้นเชิง
3.1) ตี่ / King / เรือ
3.2) เบี้ย / Pawn / จุก
3.3) สือ / เบี้ยหงาย / เม็ด
3.4) กือ / เผ่า / Rook
4. หมากรุกทั้ง 3 ชาติมีกฎการเดินซ้ำที่ใกล้เคียงกันโดยส่วนใหญ่จำกัดไว้ให้เดินซ้ำไม่เกินกี่ครั้ง
4.1) 2 ครั้ง
4.2) 3 ครั้ง
4.3) 5 ครั้ง
4.4) 7 ครั้ง
5. กลุ่มหมากชนิดใดมักมีจำนวนมากที่สุดในเกมหมากรุกเสมอ
5.1) กลุ่มของม้า (ม้า / Knight / เบ๊)
5.2) กลุ่มของเบี้ย (เบี้ย / Pawn / จุก)
5.3) กลุ่มของเรือ (เรือ / Rook / กือ)
5.4) กลุ่มของโคน (โคน / Bishop / เฉีย)
6. กฎข้อใดในหมากรุกสากล เทียบเคียงได้กับการนับศักดิ์ในหมากรุกไทย และหมากรุกจีน
6.1) กฎการเดินครบ 50 ครั้ง
6.2) กฎการเดินซ้ำไปมาสามครั้ง
6.3) กฎการเข้าป้อม
6.4) กฎการมีหมากไม่เพียงพอ
7. การเลื่อนขั้น / เลื่อนยศหมากกลุ่มเบี้ย ข้อใดกล่าวโดยสรุปคร่าวๆ ได้ถูกต้อง
7.1) แนวเลื่อนขั้นมักอยู่หลังเดินผ่านครึ่งกระดานไปแล้วเสมอ
7.2) การเลื่อนขั้นจะบังคับเลื่อน และไม่สามารถลดขั้นกลับไปได้
7.3) ยิ่งแนวเลื่อนขั้นอยู่ไกล หมากที่ได้รับการเลื่อนขั้นยิ่งเก่งขึ้น
7.4) ทุกข้อกล่าวได้ถูกต้อง
8. การเสมอเนื่องจากรุกไม่จนเพราะมีหมากไม่เพียงพอ นอกจากหมากกลุ่มขุนแล้ว ยังมีกลุ่มใดที่หากมีตัวเดียวก็ไม่เพียงพอให้รุกจน
8.1) กลุ่มของเบี้ย (เบี้ย / Pawn / จุก)
8.2) กลุ่มของม้า (ม้า / Knight / เบ๊)
8.3) กลุ่มของโคน (โคน / Bishop / เฉีย)
8.4) กลุ่มของเม็ด (เม็ด / Queen / สือ)
9. หมากคู่ใดที่มีวิธีการเดินเหมือนกัน แต่วิธีการกินหมากต่างกัน
9.1) เบี้ย / เบี้ยหงาย
9.2) Queen / King
9.3) จุกที่ข้ามแม่น้ำแล้ว / ตี่
9.4) เผ่า / กือ
10. หมากจน หรือเข้าตาอับ (Stalemate) ในหมากรุกประเภทใด นับว่าเป็นการแพ้
10.1) หมากรุกสากล
10.2) หมากรุกไทย
10.3) หมากรุกจีน
10.4) ทุกหมากนับว่าเป็นการเสมอ
